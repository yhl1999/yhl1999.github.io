原题链接

题目大意

有$n$个商店,每个商店有一个数字$a_i$.

Dora每天都会去一些商店,在这一天里Swiper将会去剩下的那些Dora没去过的商店.

如果这一天里Dora去的所有的商店对应数字的最小公倍数,大于Swiper去的所有商店对应数字的最小公倍数, 则代表这一天是Dora胜利了.

询问对于接下来的m天里,给定每天Dora会去的商店的编号,是否存在一个数列$a_i$使Dora每天都能胜利.

解题思路

lcm(最小公倍数)

  • 可通过分解两数的质因数,最小公倍数等于它们所有质因子的乘*积,如果有相同的质因子则取出现次数较多的.
  • 两数的乘积 = 两数的最小公倍数与最大公因数的乘积

性质:
$$\text{对于整数集A B}，lcm(B)\le lcm(A)\quad B\subset A$$

定义$D_i$代表Dora第$i$天所去商店对应的数集,$S_i$为Swiper第$i$天所去商店对应的数集.

首先我们考虑不存在$a_i$满足条件的情况:如果存在某两天$i$与$j$满足$D_i$与$D_j$完全不重合则不存在满足条件的数组$a_i$,因为如果$D_i$与$D_j$不重合则容易得知$D_j\subset S_i\quad D_i\subset S_j$,由前面的性质可以得知$lcm(D_i)\le lcm(S_j)\quad lcm(D_j)\le lcm(S_i)$

假设有$lcm(D_i)>lcm(S_i)$则有$lcm(D_i)>lcm(S_i)\ge lcm(D_j)$

若$lcm(D_j)>lcm(D_j)$则又可推出$lcm(D_j)>lcm(S_j)\ge lcm(D_i)$

所以必不可能存在数集$a_i$满足条件.

接下来考虑$a_i$存在的情况,首先生成$m$个不同的素数$p_1,p_2,…,p_m$,那么$a_i=\prod p_j$,$j$代表Dora在第$j$天访问商店$i$.

证明:

We know that on day $i$, Dora bought an integer from a store that she also visited on day $j$, and this number must be a multiple of $p_j$. So $lcm(Di)=p_1p_2…p_m$ for all $i$. On the other hand, $S_i$ contains no multiples of $p_i$, because they are all in $D_i$. So the $lcm$ of $S_i$ is strictly smaller.
Now we just need to check that any two days have a common store, which can be done in $O(nm2)$ by checking each pair of days and determining for each $i$ whether Dora visited both stores on day $i$ in $O(n)$. You can achieve a slight speedup if you check this using a bitset, but this wasn’t necessary to solve the problem.
